┌─────────────────────────────────────────────────────────────┐
│ ЕТАП 1: NAMECHEAP (5 хвилин)                                │
└─────────────────────────────────────────────────────────────┘

1. Зайти на: https://www.namecheap.com
2. Domain List → zelma4.me → Manage
3. Advanced DNS → Add New Record

   ┌──────────────────────────────────────────┐
   │ Type:  A Record                          │
   │ Host:  bunker                            │
   │ Value: ВАШ_IP_СЕРВЕРА (123.45.67.89)    │
   │ TTL:   Automatic                         │
   └──────────────────────────────────────────┘

4. Натисніть зелену галочку ✓

⏰ Зачекайте 5-30 хвилин для DNS propagation


┌─────────────────────────────────────────────────────────────┐
│ ЕТАП 2: ПЕРЕВІРКА DNS (опціонально)                         │
└─────────────────────────────────────────────────────────────┘

На своєму комп'ютері:
$ dig bunker.zelma4.me
або
$ nslookup bunker.zelma4.me

Має показати IP вашого сервера ✓


┌─────────────────────────────────────────────────────────────┐
│ ЕТАП 3: НАЛАШТУВАННЯ СЕРВЕРА (автоматично)                  │
└─────────────────────────────────────────────────────────────┘

Підключіться до сервера:
$ ssh user@your-server-ip

Завантажте проект:
$ git clone https://github.com/zelma4/bunker-game
$ cd bunker-game

Запустіть автоматичний скрипт:
$ sudo ./setup-domain.sh

Скрипт автоматично:
✓ Встановить Nginx
✓ Налаштує конфігурацію для bunker.zelma4.me
✓ Встановить Certbot
✓ Отримає безкоштовний SSL сертифікат
✓ Налаштує firewall (порти 80, 443)
✓ Налаштує автоматичний redirect HTTP → HTTPS


┌─────────────────────────────────────────────────────────────┐
│ ЕТАП 4: ЗАПУСК ГРИ                                          │
└─────────────────────────────────────────────────────────────┘

$ ./deploy-server.sh

Додаток запуститься на localhost:8765
Nginx проксуватиме запити з HTTPS

┌─────────────────────────────────────────────────────────────┐
│ КОРИСНІ КОМАНДИ                                              │
└─────────────────────────────────────────────────────────────┘

Переглянути логи Nginx:
$ sudo tail -f /var/log/nginx/bunker.zelma4.me.error.log

Перезапустити Nginx:
$ sudo systemctl restart nginx

Переглянути статус Docker:
$ docker-compose ps

Переглянути логи гри:
$ docker-compose logs -f

Оновити SSL сертифікат:
$ sudo certbot renew


┌─────────────────────────────────────────────────────────────┐
│ TROUBLESHOOTING                                              │
└─────────────────────────────────────────────────────────────┘

502 Bad Gateway?
→ Перевірте: curl http://localhost:8765
→ Переглянути: docker-compose logs

DNS не працює?
→ Зачекайте 30 хвилин
→ Перевірте A-запис на Namecheap

SSL не працює?
→ sudo certbot --nginx -d bunker.zelma4.me --force-renewal
